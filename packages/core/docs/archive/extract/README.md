Extract was the prototype-reconstruction research track: ingest a weakly documented demo, produce a spec pack, generate an oracle, and rebuild against it.

It is retired from the live autoship product. The archived files here are kept for institutional memory and future research, not as current operating docs.

Archived implementation:

- `implementation/agents/` — old extract agents
- `implementation/skills/` — old extract skills
- `implementation/autoship.sh` — old shell runner

Agent and skill Markdown files in `implementation/` use a `.md.txt` suffix so the docs site does not publish retired runnable prompts as current documentation.
